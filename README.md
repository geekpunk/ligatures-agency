Website for the *Agency* LP by Ligatures. Built with React + Vite and deployed to GitHub Pages.

## Features

- Flippable album cover (click to toggle front/back)
- HTML5 audio player with full playlist playback
- Track-level download links
- Shareable URLs via `?song=` query string (e.g. `?song=white_rose` or `?song=B3`)
- Links to [Bandcamp](https://ligatures.bandcamp.com/), [Instagram](https://www.instagram.com/ligatures.dc), and [GitHub](https://www.github.com/geekpunk/ligatures-agency)

## Setup

```
npm install
npm run dev
```

## Build

```
npm run build
```

## Deploy

Pushes to `main` auto-deploy via GitHub Actions. Alternatively:

```
npm run deploy
```

## Song files

Place MP3s in `public/songs/`:

- `all_kindsa_guys.mp3`
- `white_rose.mp3`
- `eyes_wide_open.mp3`
- `currency_musuem.mp3`
- `position_paper.mp3`
- `sheer_terror.mp3`
- `naval_observatory.mp3`
